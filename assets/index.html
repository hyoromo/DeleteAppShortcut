Method of setting up widget
  1. It is a long push in the Home screen.Or, Please choose [menu button] -> [Add].
  2. "Add to Home screen" Because the screen is displayed, Please choose "Shortcuts".
  3. "Choose shortcuts" Because the screen is displayed, Please choose "DeleteAppShortcut".

How to use
  1. The application program list is displayed when this application program is started, and select the application program that wants to make the deletion short cut, please.
  2. After it arranges it in Home, the uninstallation screen starts the short cut icon when touching.
  3. Please uninstall it according to a standard uninstaller.

Help
    Please relocate the widget when operation becomes amusing.
    Still, please reinstall the application program when it is amusing.
